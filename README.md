---
### Requirements
        Pygame
---
#### Controls
        ESC        -> QUIT(close window)
        MOUSECLICK -> Instantiate Sand (spawn sand particles by maintaining the left mouse click)
        R          -> Restart(Reset)
        SPACE      -> Pause the simulation
